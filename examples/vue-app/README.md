# vue-app

## Create a local build of tKey
```
yarn
yarn bootstrap
yarn pack:lerna
```

## Project setup
```
npm install:local
npm install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
